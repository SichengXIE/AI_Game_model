# AI Game Mod Studio

AI-assisted game mod and placeable-asset creation platform.

The initial target is `Cities: Skylines II`, with the product direction focused on:

- `Base Mod + Spec Runtime`
- placeable station and airport templates
- AI-assisted asset design, validation, and sharing

## Repository Layout

- `docs/`: product plans and architecture notes
- `frontend/`: web app
- `backend/`: API and orchestration services
- `base-mod/`: in-game runtime mod
- `schemas/`: shared spec schemas
- `template-library/`: reusable station and airport templates
- `.github/`: repository automation
